# 🚀 AI Resume Analyzer & ATS Optimizer

An AI-powered web application that analyzes a user's resume against a target job description and provides an ATS (Applicant Tracking System) compatibility score along with improvement suggestions.

## 📌 Project Overview

The AI Resume Analyzer & ATS Optimizer helps job seekers improve their resumes by comparing them with a job description. It identifies missing skills, analyzes keyword matching, and provides suggestions to increase ATS compatibility and interview chances.

## ✨ Features

- 📄 Upload Resume (PDF)
- 🎯 Paste Target Job Description
- 🤖 AI-powered Resume Analysis
- 📊 ATS Match Score
- 🔍 Keyword Matching
- 💡 Resume Improvement Suggestions
- ⚡ Responsive User Interface
- 🎨 Modern & Clean Design
